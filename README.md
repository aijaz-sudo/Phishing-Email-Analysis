# 🔥 Phishing Email Analysis – SOC L1 Project

## 🎯 Objective

This project demonstrates a phishing email investigation workflow performed from a SOC Analyst L1 perspective.

The investigation includes:
- Email analysis
- Header analysis
- Threat intelligence correlation
- IOC extraction
- Incident reporting

---

# 🛠 Tools Used

- VirusTotal
- URLScan.io
- AbuseIPDB
- PhishTank
- WHOIS ICANN Lookup
- Google Header Analyzer
- VS Code
- GitHub

---

# 🔍 Investigation Workflow

1. Created phishing email sample
2. Analyzed phishing indicators
3. Investigated email headers
4. Performed SPF/DKIM/DMARC analysis
5. Investigated malicious URL reputation
6. Investigated sender IP reputation
7. Performed WHOIS domain analysis
8. Extracted Indicators of Compromise (IOCs)
9. Created incident response report
